===
Small (micro) virtual game console (fantasy console).

## Specifications
- Custom assembly language
- 48Kb of RAM
- 8 registers
- 120x96 screen

## Building
Just type
```
$ make
```

## Usage
```
$ ucpu <binary file> or <assembly code file>
```

See test.ASM for an example.
